User

Get Token

POST /login/

Permission: Registered user with Token

Parameter

Field Type Description
username String

The Username.
password String

The Password.
  • Usage (example): 
POST /login/ HTTP/1.1
Content-Type: application/json
Host: 127.0.0.1:8000
Connection: close
User-Agent: Paw/2.1.1 (Macintosh; OS X/10.10.2) GCDHTTPRequest
Content-Length: 35
    {
        
            "username":"dor",
            "password":"123"
    }
  • Response (example): 
"{ 'Token':'ec1db856e230a61cc12ad5040a554c2a312fcee9'}"

Add Incentive

POST /api/incentive/

Permission: Registered user with Token

Parameter

Field Type Description
schemeName String

The Scheme Name.
schemeID Int

The Scheme ID.
text String

The Text of the Incentive.
typeID Int

The Type ID.
typeName String

The Type Name.
status Boolean

The Status.
ordinal Int

The ordinal number.
tags Tag[]

List of Tags for this Incentive (Array of Tags).

modeID Int

The Mode-ID.
groupIncentive Boolean

If this a group Incentive.
condition String

The Condition for this incentive.
  • Usage (example): 
POST /api/incentive/ HTTP/1.1
Authorization: Token ec1db856e230a61cc12ad5040a554c2a312fcee9
Content-Type: application/json
Host: 0.0.0.0:8000
Connection: close
